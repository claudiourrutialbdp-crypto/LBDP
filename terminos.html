Condiciones de Uso del Sitio Web

Última actualización:

1. Aceptación de los Términos

Al acceder y utilizar el sitio web del Liceo (en adelante, "el Sitio"), usted acepta estar sujeto a estos Términos y Condiciones de Uso, todas las leyes y regulaciones aplicables, y acepta que es responsable del cumplimiento de las leyes locales aplicables.

Si no está de acuerdo con alguno de estos términos, tiene prohibido usar o acceder a este sitio. Los materiales contenidos en este sitio web están protegidos por las leyes de derechos de autor y marcas comerciales aplicables.

2. Naturaleza del Sitio Web

Este es un sitio web informativo del Liceo que proporciona:

Lo que SÍ incluye:

  • Información institucional del Liceo
  • Oferta educativa y programas
  • Calendario de actividades
  • Documentos descargables
  • Formulario de contacto
  • Mapa de ubicación

Lo que NO incluye:

  • Foros de discusión
  • Comentarios públicos
  • Cuentas de usuario o registro
  • Interacción entre visitantes
  • Contenido generado por usuarios
  • Plataforma educativa online
Clarificación: Este sitio web es únicamente informativo y de consulta. No es una plataforma interactiva ni cuenta con funcionalidades de red social o comunidad online.

3. Licencia de Uso

3.1. Permisos Otorgados

Se le otorga permiso para:

  • Navegar libremente por todas las páginas públicas del Sitio
  • Visualizar e imprimir páginas del Sitio para su referencia personal
  • Descargar documentos públicos disponibles (calendarios, información de uniformes, etc.)
  • Usar el formulario de contacto para consultas legítimas
  • Acceder a información sobre los servicios educativos del Liceo

3.2. Restricciones

Esta licencia NO le permite:

Prohibiciones:
  • Modificar o copiar los materiales para uso comercial o distribución
  • Usar el contenido, imágenes o textos del Sitio para fines comerciales sin autorización
  • Eliminar cualquier derecho de autor, marca registrada u otras notaciones de propiedad
  • Republicar o redistribuir el contenido del Sitio sin permiso expreso
  • Usar herramientas automatizadas (bots, scrapers) para extraer contenido masivamente

4. Propiedad Intelectual

Uso Autorizado

El uso de contenido solo está permitido según lo establecido en estos términos

5. Formulario de Contacto

El Sitio cuenta con un formulario de contacto que permite enviar consultas directamente al Liceo. Al usar este formulario, usted garantiza que:

  • La información proporcionada (nombre, email, teléfono, mensaje) es verdadera y precisa
  • El contenido de su mensaje es apropiado y no viola ninguna ley
  • No enviará spam, mensajes ofensivos, amenazantes o fraudulentos
  • Su mensaje no contiene contenido malicioso, enlaces engañosos o intentos de phishing
Importante: Los datos del formulario se envían mediante EmailJS directamente a nuestro correo institucional. No se almacenan en ninguna base de datos. El Liceo responderá a las consultas legítimas en un plazo razonable.

6. Uso Apropiado del Sitio

Al navegar y utilizar este Sitio web informativo, usted se compromete a NO:

Enviar spam o mensajes masivos a través del formulario de contacto
Intentar acceder a áreas administrativas o restringidas del Sitio
Interferir con el funcionamiento normal del Sitio mediante ataques DDoS o similares
Usar herramientas automatizadas para extraer contenido (scraping) sin autorización
Intentar comprometer la seguridad del Sitio mediante hacking o inyección de código
Usar el formulario de contacto para actividades fraudulentas, phishing o engaños
Importante: Este es un sitio web informativo sin interacción entre usuarios. No hay foros, comentarios públicos ni cuentas de usuario. El único medio de contacto es el formulario que envía mensajes directamente al Liceo.
